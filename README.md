# 📊 Customer Behavior Analysis Project

## Overview

This project demonstrates an **end-to-end data analytics workflow**.
The goal was to transform raw data into meaningful insights by using multiple tools and techniques including **Python, SQL Server, and Power BI**.

The project covers the full analytics process: **data loading, exploratory data analysis (EDA), data cleaning, querying, visualization, and reporting**.

---

## Dataset

The dataset used in this project contains transactional and customer-related information that allows analysis of patterns, trends, and performance metrics.

Key characteristics of the dataset:

* Multiple columns related to customer activity and purchases
* Structured tabular format
* Used for analysis, querying, and dashboard visualization

---

## Tools & Technologies

* **Python** – Data loading, cleaning, and exploratory data analysis
* **Pandas & Matplotlib/Seaborn** – Data manipulation and visualization
* **SQL Server** – Data querying and analytical SQL operations
* **Power BI** – Interactive dashboard and visual analytics
* **Gamma** – Presentation creation
* **GitHub** – Project documentation and version control

---

## Project Steps

### 1. Data Loading

The dataset was loaded into **Python using Pandas** to begin the analysis process.

### 2. Exploratory Data Analysis (EDA)

EDA was performed to understand the dataset by:

* Checking data types
* Exploring distributions
* Identifying missing values
* Detecting potential patterns or anomalies

### 3. Data Cleaning

Data preprocessing steps included:

* Handling missing values
* Fixing inconsistent formats
* Removing duplicates
* Preparing the data for analysis and SQL queries

### 4. SQL Analysis

The cleaned dataset was imported into **SQL Server** where analytical queries were performed, including:

* Aggregations
* Filtering
* Ranking
* Customer segmentation
* Performance metrics calculations

### 5. Dashboard Creation

An **interactive Power BI dashboard** was built to visualize the insights and key metrics using:

* KPI cards
* Charts and graphs
* Filters and slicers for dynamic exploration

### 6. Reporting

A **data analysis report** was created summarizing:

* Key findings
* Business insights
* Analytical results

### 7. Presentation

A **PowerPoint presentation was generated using Gamma** to present the project in a clear and structured way.
